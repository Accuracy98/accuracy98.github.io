IF线社会主义中国小说设定草案

《未竟之火》草案（修订版）

核心定位

这是一部长篇架空历史小说。

分叉点设定：
1976 年，毛泽东没有去世。身体一度极度虚弱，但在全国普遍认为其将离世的时刻，他意外恢复，重新介入党、军队与国家的最高决策。中国由此没有沿着现实历史进入改革开放，而是进入一条新的道路：在吸收文革失控教训后，将“无产阶级专政下继续革命”逐步制度化，形成一种不走邓式市场化、也不简单重复原样文革的社会主义中国。

叙事原则

  1. 不把小说写成政治口号堆砌，而要写成有真实人物、真实犹豫、真实代价的历史叙事。
  2. 不把任何阵营脸谱化。即使是“走资派”，也要写出其效率、秩序、恢复生产的吸引力；即使是“继续革命派”，也要写出其对官僚化和复辟的真实恐惧。
  3. 小说的张力，不来自“谁更会喊口号”，而来自：谁能让这个国家活下去，谁又能避免它变质。
  4. 主线是国家命运，落点却必须在人身上：干部、工人、青年、军人、知识分子、农村基层，都要有自己的命运。
  5. 毛活下来不意味着一切自动变好。真正的主题不是“神迹救国”，而是：一个原本可能滑向翻案或再度失控的国家，能否在毛晚年的最后一段时间里，被重新按回一条更难、更窄、但也更可持续的社会主义道路。
  6. 小说里所有真实历史人物，都尽量贴近其真实政治气质、语言习惯和能力边界；不把他们写成现代人口吻的传声筒。

故事总结构

第一部：余火（1976—1978）

主题：主席未死，原本已经悄悄开始的“后毛时代”被突然中止。

核心冲突

  • 毛泽东病重后恢复，震动整个政治格局。
  • 华国锋已经被推到正式接班位置，却发现自己的“稳”并不等于真正定局。
  • 叶剑英首先关心的是军队边界和中央稳定，但其深层政治判断更接近邓式整顿逻辑。
  • 邓小平虽然被压下，但其“整顿—恢复秩序—专家归位”的路线吸引力并未消失。
  • 江青、张春桥、王洪文、姚文元并未退场，而是在重新判断：主席重新问事后，上海经验和继续革命能否被重新制度化，而不是继续停留在运动形态。
  • 军队、地方、工厂、农村都在观望：国家到底走向右的翻案，还是走向左的失控，抑或逼出第三条路。

这一部要解决的问题

  • 毛为什么决定不再简单延续原样文革。
  • 毛如何区分“继续革命的目标”与“文革失控的形式”。
  • 为什么上海经验值得保留，但不能原样照搬。
  • 第一批新制度如何萌芽：制止武斗、限制特权、群众监督常态化、军队退出派性政治、工厂与农村的初步新章法。
  • 毛如何开始有意识地从“寻找单一接班人”转向“构建集体接班结构”。

第一部的政治格局

这一部不是两线对峙，而是至少四股力量同时在场：

  • 毛泽东：重新按住全盘，反对右的翻案，也反对左的原样失控。
  • 华国锋—汪东兴—部分中央日常系统：强调稳定、程序和中央运转。
  • 叶剑英—军队稳定派：强调军队正规化、边界与防失控，深层更认同整顿逻辑。
  • 江青—张春桥—王洪文—姚文元：强调防翻案、上海经验、工人进政权与继续革命路线，但内部也并不完全一致。
  • 邓小平及其影响力：本人受限，但其路线仍持续作为现实高效方案诱惑党内和地方。

主要人物

毛泽东：晚年但未失判断力。开始意识到，若继续依赖个人威望发动运动，社会主义终将失控；若完全交给整顿派，则可能走向翻案。他的最终课题是：把继续革命从运动形态转为制度形态。

华国锋：正式接班位置上的“中间人”。忠诚、谨慎、重程序、重稳定，行政能力不低，但理论创造力有限。他是国家机器的承重梁，却不是天然的路线领袖。

叶剑英：军队与元老系统中的稳定器。现实、克制、重边界、重秩序，不愿军队再卷入地方和派系斗争；深层政治判断更接近邓式整顿，但在毛仍在时必须先服从“稳住中央、防止失控”的大局。

邓小平：极强的现实政治家，相信没有秩序和生产恢复，一切社会主义理想都只是空谈。他不是“纯粹反派”，而是最具威胁也最有说服力的另一条道路代表。

张春桥：四人帮中最有可能被保留下来并参与未来制度设计的人。冷、整、强逻辑，擅长把上海经验、工人进政权和防修路线写成制度语言。

江青：锋利、压场、斗争性极强。她的直觉常常准确，但容易把斗争推向人格化和宗派化。适合做路线监督者，不适合独自统国家机器。

王洪文：工人进入最高权力中心的象征人物。直、急、带工人本能与现实感，潜力很大，但现实历史中的问题是“拔得太快、练得太少”。在 if 线里，他有成长为重要工人领袖的可能，但未必成为最高统治者。

姚文元：宣传和理论文字的刀锋人物。擅长修辞、批判和口径塑形，但不适合主政。

陈云：冷静、审慎，看重经济规律和农村实际，主张在集体框架内恢复弹性，是连接毛晚年制度修正与基层实际的重要人物。

虚构主角一：沈砚秋
北京某中央机关青年秘书，出身普通工人家庭，文革中崇拜过造反，如今开始第一次真正接触国家高层运转。他是读者进入中枢的眼睛。

虚构主角二：顾北川
东北国营大厂车间技术员，经历过文革中“专家靠边站”的羞辱，也经历过工人管理带来的短暂光亮。他是工业线的代表人物。

虚构主角三：赵荷生
华东农村生产队妇女骨干，养猪、记工分、组织妇女劳动样样拿手。她将见证“人民公社 2.0”的形成。

虚构主角四：林霁
军队青年参谋，出身革命家庭，在“党指挥枪”教育中长大，却亲眼见过军队卷入地方斗争后的混乱。他是军队线的代表。

第二部：定盘（1978—1984）

主题：制度化继续革命开始成形，旧阵营被打散，新章法被一点点搭起来。

核心内容

  • 党的路线之争公开化，但不再通过全面打倒解决，而是通过中央全会、试点、群众监督和有限组织斗争推进。
  • 农村不搞包产到户，而是形成“生产队基本核算 + 公社统筹服务 + 责任细化到组到人 + 公私并举养殖”的新结构。
  • 工业系统开始形成修正版鞍钢路线：工人参与管理、技术重新归位、干部下厂、规章改革试点、革命委员会经常化。
  • 军队职业化建设启动，明确不得卷入派系政治，不得延伸任务边界。
  • 上海经验被重新吸收：不是原样复制“上海人民公社”，而是把工人进政权、群众监督干部和三结合推进为经常性制度。
  • 接班人制度从“钦定一人”改成“集体接班 + 梯队培养”。
  • 继续引进西方技术，但强调不让资本控制命脉产业。
  • 王洪文被有意识地“再培养”，从工人象征人物向更成熟的工业—城市领导者推进。

这一部的戏剧冲突

  • 地方出现隐性“单干潮”和“包田呼声”。
  • 工业部门中一部分人主张完全恢复专家治厂。
  • 继续革命派内部也出现分歧：到底是要保留运动冲击，还是接受制度化收束。
  • 华国锋、张春桥、叶剑英之间围绕“稳—改—防乱”的节奏不断出现分歧。
  • 邓式路线虽然未能翻盘，但作为高效率方案持续诱惑干部系统。
  • 西方对中国的观望开始转为试探：这个中国究竟会不会像现实那样走向市场化？

第二部要完成的关键转折

  • 毛最终确认：后毛时代不能再设单一储君。

  • 形成“后毛四柱”的雏形：

    • 张春桥：路线与制度
    • 华国锋：行政与国家日常运转
    • 叶剑英：军队与稳定边界
    • 王洪文：工人政治与城市工业代表性上升通道

第三部：大国新形（1984—1992）

主题：社会主义中国开始长出自己的现代化形态，毛逐渐老去，后毛秩序第一次被认真准备。

核心内容

  • 四三方案式技术引进升级为国家工业现代化战略。

  • 半导体、计算机、数控机床、通信、军工电子成为国家攻关核心。

  • 城乡关系未按现实路线剧烈撕裂，农村仍保持较强集体性。

  • 对外贸易扩大，但不按 WTO 式深度市场化改造自己。

  • 苏联改革、东欧动荡传入中国，引发全党对“社会主义是否还能现代化”的巨大争论。

  • 毛的身体开始真正不可逆衰老，接班问题从潜在议题变成现实议题。

  • 四柱结构内部逐渐分工清晰，但分歧也越来越明显：

    • 华国锋更重行政稳定
    • 张春桥更重路线和制度纯度
    • 叶剑英更重军队和国家边界
    • 王洪文更重工人政治与工业治理

核心戏剧点

  • 苏联危机与东欧变化震动中国，给邓式路线提供新的“现实理由”。
  • 四人接班结构是否真能取代单人继承，第一次接受压力测试。
  • 毛最终去世，后毛时代正式开启。

第三部的关键结局设想

  • 毛去世后，不设单一最高继承人。

  • 形成集体接班：

    • 张春桥成为最接近“路线第一继承人”的人物。
    • 华国锋继续承担国家行政中枢。
    • 叶剑英继续作为军队边界和稳定器。
    • 王洪文成为工人政治与城市工业路线的重要代表。

  • 江青与姚文元继续保有影响力，但不进入“四柱”核心分工。

  • 邓小平路线没有消失，而成为后毛时代最大的长期压力来源。

第四部：孤与不孤（1992—2008）

主题：在全球资本主义胜利叙事中，坚持自己的道路；后毛集体领导接受真正历史检验。

核心内容

  • 苏联解体后，中国成为世界最大社会主义国家，也成为西方真正的长期对手。
  • 中国没有被全面孤立，但面对更强技术封锁与舆论围堵。
  • 中国通过超大市场、国家产业政策、全球南方合作，建立不同于现实世界的全球关系网络。
  • 国内开始争论：要不要更深加入全球贸易体系？是否为了技术必须进一步开放？
  • 邓式路线在后毛时代并未消失，反而以“苏东剧变后的现实主义答案”重新吸引一部分干部、专家与地方力量。
  • 四柱集体领导内部出现制度性分歧，但军队边界与反改线高门槛机制仍把它们压在社会主义框架内。

戏剧点

  • 年轻一代对“继续革命”语言感到疏离，但又切身享受到了低分化、高保障的社会成果。
  • 工业、军工、教育与基层公共服务系统逐渐显示出这条道路的优势，也暴露僵化与低活力问题。
  • 四柱结构开始面对代际更替：谁能继承“制度化继续革命”，而不是只继承某一个人的政治遗产？

第五部：计算与公社（2008—2025）

主题：社会主义中国进入 AI 与大系统计划时代。

核心内容

  • 中国将 AI、芯片、工业软件、全国算力网络上升为国家工程。
  • 计划经济不再主要依赖层层报表，而是结合大数据、计算机与 AI 做动态调度。
  • 人民公社 2.0 与县域工业、数字供销系统、农村合作养殖、基层医疗网络深度结合。
  • 中国成为世界头号工业—科技社会主义国家，但也面临新问题：算法官僚主义、技术精英再分层、全球冲突升级。
  • 后毛时代真正完成制度化：继续革命不再主要靠个人威望，而是靠一整套接班、监督、军队边界和反改线机制维持。

最终大主题

  • 中国没有走向现实世界那条资本主义全球化道路。
  • 也没有停留在 1966—1976 年原样文革的无限延长。
  • 它走出的是一条更难、更慢、更充满内部张力，但也更平均、更工业化、更公共化、更强调工人农民政治地位的社会主义现代化道路。

小说风格

  1. 不写成纯说明书，而要通过人物命运展开制度变迁。

  2. 语言可以有历史厚重感，但不能全是大会发言体。

  3. 每一部都要有“小人物看大时代”的镜头：食堂、工厂、会议室、农村猪圈、军营、研究所、火车站。

  4. 要让读者感到：这不是简单的“毛活下来所以一切都好了”，而是一条始终在危险边缘试图自我修正的道路。

  5. 真实历史人物要尽量贴近其历史性格与说话习惯：

    • 张春桥冷、整、逻辑强
    • 江青锋利、压场、好抢先定调
    • 王洪文直、急、工人感强，但需要成长
    • 姚文元重修辞、重措辞、重口径
    • 华国锋稳、谨慎、程序感强
    • 叶剑英现实、克制、重军队边界
    • 邓小平高效、现实、始终构成另一种诱惑性未来

  6. 这部小说的核心，不是谁“赢了”，而是：中国能否在毛之后，仍然守住一条不复辟、也不失控的社会主义道路。
